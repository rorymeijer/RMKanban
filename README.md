# RMKanban

RMKanban is a fast, offline‑first Kanban board for desktop. It keeps your boards as local files, supports flexible workflows, and offers rich exports for sharing.

## Highlights
- Offline‑first: boards live on disk and work without an account or server
- Customizable columns, swimlanes, labels, and templates
- Card details, assignees, and activity history
- Exports: plain text, PDF and more
- Designed for macOS & Windows

## Data & Privacy
- Board data is stored locally in a folder you choose.
- No cloud sync is built in.

## Getting Started
- Download the latest release from GitHub.
- Open the app, then create or open a board folder.
